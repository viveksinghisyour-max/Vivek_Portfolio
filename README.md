# Vivek Portfolio

A modern, responsive portfolio website showcasing cybersecurity expertise and projects.

## Features

- **Cybersecurity Theme**: Dark theme with cyber-inspired design elements
- **Responsive Design**: Optimized for all device sizes
- **Interactive Elements**: Smooth animations and hover effects
- **Project Showcase**: Display of featured projects with filtering
- **Contact Form**: Functional contact form for inquiries
- **Social Links**: Integration with professional social platforms

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom properties, animations, and responsive design
- **JavaScript**: Interactive functionality and form handling
- **Google Fonts**: Inter font family for typography

## Local Development

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Resume/Vivek_Portfolio
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx serve .
   ```

3. Visit `http://localhost:8000` to view the site.

## Deployment

This project is configured for deployment on Vercel:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect the static site configuration
3. Deploy with the provided `vercel.json` settings

## Customization

- **Colors**: Modify CSS custom properties in `:root` for theme changes
- **Content**: Update HTML sections for personal information
- **Projects**: Add new projects in the projects section
- **Contact**: Update contact information and social links
